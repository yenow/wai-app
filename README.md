# 🌈 wai-app
`WAI` 어플입니다. `flutter`로 만들었습니다.

## ☁ WAI 앱이란?
WHO AM I의 약자로 이름을 지었습니다. 에니어그램 커뮤니티로, 에니어그램이라는 성격유형테스트로 내가 어떤 사람인지 알고, 다른사람과 소통하면서 타인의 성향도 이해하면 좋겠다는 취지로 만들었습니다.
